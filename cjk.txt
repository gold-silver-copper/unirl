*   兔**子** (tù**zi**) - Rabbit
*   猴**子** (hóu**zi**) - Monkey
*   狮**子** (shī**zi**) - Lion
*   鸭**子** (yā**zi**) - Duck
*   蚊**子** (wén**zi**) - Mosquito

**General Description + Material + Core Noun**


---

### 1. "A horse leather left glove"

In Chinese, the order for describing a noun is:
**Left/Right + (Other attributes) + Material + Noun**

*   **Left:** 左 (zuǒ)
*   **Glove:** 手套 (shǒutào)
*   **Horse Leather:** 马皮 (mǎpí) - "Horse" + "Leather/Skin"

Putting it together, we get:
**左 + 马皮 + 手套**

> **Simplified:** 一只左马皮手套
> **Traditional:** 一隻左馬皮手套
> **Pinyin:** Yī zhī zuǒ mǎpí shǒutào.
> **Literal:** "One [classifier] left horse-leather glove."

**Explanation:**
*   **一只 (yī zhī):** "One." 只 (zhī) is a common classifier for one of a pair (like gloves, hands, eyes).
*   The order is the opposite of English: "Left" comes before the material "horse leather."

---

### 2. "Wool sweater with embroidered golden thread in cross patterns"

This is more complex and requires breaking it into logical chunks. The core noun is "sweater" (毛衣, máoyī).

The descriptive order is:
**Pattern + Design/Embellishment + Material + Noun**

Let's identify the components:
*   **Wool:** 羊毛 (yángmáo)
*   **Sweater:** 毛衣 (máoyī) - This word already contains "wool" (毛), so you often don't need to repeat it. However, for clarity and emphasis, you can.
*   **Embroidered:** 绣有...的 (xiù yǒu... de) - "embellished with..."
*   **Golden Thread:** 金线 (jīnxiàn)
*   **Cross Patterns:** 十字图案 (shízì tú'àn)

We can structure this in two main ways:

#### **Option 1: More direct and common**

We describe the sweater as a "wool sweater" and then add the embroidery as a separate clause.

> **Simplified:** 一件绣有金线十字图案的羊毛毛衣。
> **Traditional:** 一件繡有金線十字圖案的羊毛毛衣。
> **Pinyin:** Yī jiàn xiù yǒu jīnxiàn shízì tú'àn de yángmáo máoyī.
> **Literal:** "One [classifier] embroidered-with golden-thread cross-pattern **'s** wool sweater."

#### **Option 2: More concise (and very natural)**

Since 毛衣 (máoyī) implies "wool," we can drop 羊毛 (yángmáo) to avoid repetition.

> **Simplified:** 一件绣有金线十字图案的毛衣。
> **Traditional:** 一件繡有金線十字圖案的的毛衣。
> **Pinyin:** Yī jiàn xiù yǒu jīnxiàn shízì tú'àn de máoyī.
> **Literal:** "One [classifier] embroidered-with golden-thread cross-pattern **'s** sweater."

---

### **Key Grammar Takeaways:**

1.  **The Magic 的 (de):** This particle is the glue that attaches all descriptions to the main noun. Think of it as **'s**.
    *   `[绣有金线十字图案]的[毛衣]` = `[embroidered with gold thread]'s [sweater]`

2.  **Word Order is Reversed from English:** Always start with the smallest detail and move toward the main noun.
    *   **English:** Wool sweater [with] embroidered golden thread [in] cross patterns.
    *   **Chinese:** [Cross-pattern golden-thread embroidered]'s wool sweater.
    *   `十字图案 (cross pattern)` -> `金线 (gold thread)` -> `绣有...的 (embroidered with)` -> `毛衣 (sweater)`

3.  **Classifiers (量词 liàngcí):** Remember to use the correct classifier before the noun.
    *   **只 (zhī):** For one of a pair (gloves, shoes, hands).
    *   **件 (jiàn):** For upper body clothing (shirts, sweaters, jackets).

By following this structure—**Specific Description + 的 + Material + Core Noun**—you can accurately describe almost any object in Chinese.
